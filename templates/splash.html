Versioned data, collaborated

Dat is a version-controlled, distributed data tool designed to improve collaboration between data people and data systems.

Version by key or by file

Add data into dat from newline-delimited json, csv, or binary format. You can version the data by key or by file. Keys are auto-generated if no key column name is supplied.
Learn more >>

$ dat import values_per_city.csv --key=id
Added 302,143 keys (32.03 Mb, 4.4 Mb/s).
Current version is now bad31c32432c.

$ dat write model_data.tar.gz
Added model_data.tar.gz (153.03 Mb, 15.3 Mb/s).
Checked out to version cef3221asd1.
SSH and HTTP endpoints

Dat is transport agnostic, but there are clients for popular transports like http or ssh. You can also pipe it or add it to an existing server easily. Learn more >>

$ dat clone ssh://uni.edu/maxogden/flights

Clone from remote has completed.
Current version is now c2342de3f
