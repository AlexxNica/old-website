The Dat Project

We build open source tools to unlock the potential of important datasets in science, journalism and government.
Try Dat
Dat is our dataset sharing system, you can use it on the web or on the command line.
Try Dat.land in your browser
Or if you have node.js installed you can install the dat CLI tool:
npm install -g dat
Distributed Sync
Dat shares versioned data through a decentralized network, getting faster as more users use it.
Intelligent Storage
Dat deduplicates data between versions, reducing bandwidth costs and improving speed.
Use it anywhere
It works on the web, plus there’s a command-line tool, a javascript client library, and a desktop application.
