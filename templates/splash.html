Versioned data, collaborated

Dat is a version-controlled, distributed data tool designed to improve collaboration between data people and data systems.
Dat Beta now available for early adopters
$ npm install -g dat
Version by key or by file

Add data into dat from newline-delimited json, csv, or binary format. You can version the data by row or by file. Keys are auto-generated if no key column name is supplied.

> from datpy import dat
> repo = Dat('./data', dataset="cities_data")

# import data from a file on disk
> v1 = repo.add_file("cities.csv", key="cityId")
> v1
'bad31c32432c'

# get data into a pandas dataframe at given version
> df = repo.get_dataframe(checkout=v1)
> df.columns
['cityId', 'name', 'population']
SSH and HTTP endpoints

Dat is transport agnostic, but there are clients for popular transports like http or ssh. You can also pipe it or add it to an existing server easily. Learn more >>

  • Python
